# Sensor Requirements

## Camera

Purpose:
- Lane detection
- Red pillar detection
- Green pillar detection
- Parking assistance

Priority:
Critical

## Distance Sensors

Purpose:
- Wall distance measurement
- Obstacle avoidance
- Position correction

Priority:
High

## IMU Sensor

Purpose:
- Orientation estimation
- Turning stability

Priority:
Medium

## Wheel Encoder

Purpose:
- Distance estimation
- Lap counting support

Priority:
Medium
